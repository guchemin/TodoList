# TodoList API

API simples de lista de tarefas feita para aprender sobre o desenvolvimento de APIs com .NET e C#.

## Funcionalidades

- Criar tarefa
- Listar tarefas
- Atualizar status de tarefa

## Como rodar

1. Clone o repositório:
   ```bash
    git clone https://github.com/guchemin/TodoList.git
   ```
   
2. Execute:
   ```bash
    dotnet run
   ```
